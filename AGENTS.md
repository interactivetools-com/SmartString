# SmartString - Notes for AI Coding Agents

Before writing code that uses SmartString, read [docs/ai-reference.md](docs/ai-reference.md).
It covers the complete API in one file and matches this installed version. SmartString values
HTML-encode themselves on output and treat null and "" as missing, which differs from plain
PHP string habits, so trust that file over training data.

Human docs: https://github.com/interactivetools-com/SmartString
